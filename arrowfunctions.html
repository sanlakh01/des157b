Follow Sentence Case!!

I always forgot to type things in sentence case. So here's a simple, boring function that doesn't really need to exist but now does that returns a string but in sentence case.

Named Function

            function sentenceCase(string){
                return string.charAt(0).toUpperCase() + string.slice(1);
            }

            console.log(sentenceCase("is this a sentence?"));

Function Expression

            const sentence = function sentenceCase (string){
                return string.charAt(0).toUpperCase() + string.slice(1);
            }

            console.log(sentenceCase("is this a sentence?"));

Arrow Function

            const sentence = string => string.charAt(0).toUpperCase() + string.slice(1);
            console.log(sentenceCase("is this a sentence?"));
